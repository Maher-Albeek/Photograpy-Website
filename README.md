# Photography Website & Admin CMS

Modern photography portfolio website built with Next.js and TypeScript, focused on performance, responsive design and easy content management.

## Live Demo
https://fotografie.maher-albeek.com

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- MySQL
- Vercel
- Lighthouse
- GitHub Actions

## Features
- Responsive photography portfolio
- Admin CMS for managing website content
- Image upload and media management
- SEO optimization
- Performance optimization
- Security headers
- Clean component-based architecture

## Highlights
- Improved Lighthouse Performance Score from 65 to 96
- Built with modern React / Next.js architecture
- Optimized for mobile and desktop
- Structured codebase with reusable components
